UX/UI Design: Lamborghini Urus website concept


The landing page for the Lamborghini Urus is an interactive website designed to showcase the stunning features of this model. By allowing users to customize the car's color, the website seamlessly updates the color theme of the entire site, providing an immersive experience. To enhance the website further, consider adding more interactive features such as wheel and interior color options, along with high-quality images and videos that showcase the car's performance and design.
